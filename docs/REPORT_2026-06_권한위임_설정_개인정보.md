# 리테일메이트 보완 보고서 — 개인정보 · 권한위임(원격관리) · 설정/알림/더보기

작성: 2026-06 · 대상: 사장 직접관리 + 매니저 전권 위임(다매장 원격관리) 기준

---

## 0. 이번 배포에 반영된 것

- **전 직군 프로필 사진**: 사장·매니저는 [설정 → 계정 관리]에 이미 사진 업로드가 있었고, **직원(`내 정보` 화면)에도 사진 업로드/표시를 신규 구현**했습니다. 등록한 사진은 사이드바·챗봇·근무 현황에 실시간 반영됩니다.
- **근무 현황 사진**: 근태 탭에서 근무 중 직원에 등록 사진이 표시되도록 이미 반영(이전 단계).

→ 결과적으로 "사장·매니저·직원 모두 사진 등록 / 앱 진입 시 본인 사진 / 근무탭 실시간 사진" 요구는 **충족**됩니다.

## 1. 개인정보 관리 — 남은 보완점

| 항목 | 현황 | 보완 권장 | 우선 |
|---|---|---|---|
| 프로필 사진 | 전 직군 가능(반영됨) | — | 완료 |
| 회원탈퇴/계정삭제 | 없음 | **계정 삭제 + 데이터 파기 흐름**(법적·스토어 정책상 필수) | P0 |
| 개인정보 동의 관리 | 가입시 1회 | 동의 내역 조회·철회 화면(GPS·마케팅 등) | P1 |
| 내 데이터 내보내기 | 없음 | 개인정보 다운로드(요청권 대응) | P2 |
| 이메일 변경 / 휴대폰 재인증 | 없음/부분 | 계정 탭에 변경·재인증 추가 | P2 |
| 직원이 보는 본인 정보 범위 | 급여·계약 노출 | 적정(본인 한정) — 유지 | OK |

## 2. 사장 ↔ 매니저(전권 위임) 원격관리 — 보완 기능

**현황**: 사장이 직원을 "매니저로 임명"하면 매니저는 **사장과 동일한 관리 권한**을 갖고 해당 매장을 원격 관리할 수 있습니다(임명/해임, 매장 전환 지원). 위임의 기본 골격은 동작합니다.

**그러나 "전권 위임 + 원격관리"를 신뢰성 있게 운영하려면 다음이 비어 있습니다:**

| 보완 기능 | 이유 | 우선 |
|---|---|---|
| **owner 전용 안전장치** | 현재 매니저=사장과 100% 동일 권한. 매니저가 매장 삭제·다른 매니저 임명·소유권 변경까지 가능하면 위험. **민감 작업(매장 삭제, 매니저 임명/해임, 결제·소유권 이전)은 owner 전용**으로 분리 필요 | P0 |
| **활동 감사 로그(audit log)** | 위임 시 "누가 무엇을 바꿨는지"(매출 수정·직원 해고·급여 변경) 기록이 없으면 원격 위임 신뢰 어려움. 행위자·시각·대상 로그 | P0 |
| **매니저 직접 초대** | 지금은 근로계약 서명으로 가입한 직원만 매니저로 승격 가능. **휴대폰/이메일로 매니저를 바로 초대**하는 흐름 필요(원격 채용) | P1 |
| **다매장 통합 대시보드** | 매장 전환(StoreSwitcher)은 있으나 **전 매장 합산 매출·비용·인건비·근무 현황**을 한 화면에서 보는 뷰가 없음. 다매장 사장 핵심 니즈 | P1 |
| **매장별 담당 매니저 관리 화면** | 어느 매장을 누가 운영 중인지 한눈에 보는 관리 화면 | P2 |
| **권한 세분화(옵션)** | 매니저에게 "매출만/직원만" 등 부분 권한 부여 옵션(완전 위임이 부담스러운 사장용) | P2 |

## 3. 설정 · 알림 · 더보기 재검증

### 3-1. 알림 설정 — **가장 큰 공백**
- 현재 [설정 → 알림]의 4개 토글(비용 미입력/출퇴근/공지/중요)은 **선호값만 DB에 저장될 뿐, 실제로 알림을 보내는 전달 체계가 전혀 없습니다.** (웹푸시/FCM/이메일/문자/스케줄러 미구현)
- 보완(P0): **웹푸시(PWA Push) + 일일 스케줄러**로 토글에 맞춰 실제 발송. 예) 비용 미입력 저녁 알림, 직원 출근/미출근 실시간 알림, 공지 등록 시 직원 푸시. SMS(SOLAPI) 연동은 이미 있으므로 중요 알림은 문자로도 가능.

### 3-2. 매장 위치(PC/모바일)
- 주소 검색 + GPS 좌표 + 출퇴근 반경 — **정상**. 
- 보완(P2): 지도 미리보기/핀 미세조정, 반경 시각화(원) 표시.

### 3-3. 개인설정(계정)
- 사진·이름·전화·비밀번호 — 동작. 
- 보완: 계정삭제(P0)·이메일변경(P2)·기기 로그아웃/세션 관리(P2)·2FA(P3).

### 3-4. 모바일 "더보기" 탭
- 현재: 비용/리포트/근태/계약/공지/설정. 
- 보완(P2): **내 정보(프로필)·매장 전환·도움말/문의·로그아웃**을 더보기에서도 접근 가능하게(현재 로그아웃·프로필은 우상단에만 있음). 매니저/직원은 노출 항목 분기.

### 3-5. PC 세부 개인설정
- 설정 탭(매장/계정/알림/급여/보안) 구성은 양호. 보안 탭은 비밀번호 변경만 → 세션·2FA 보강(P2~P3).

---

## 4. 권장 진행 순서

1. **P0 — 알림 전달 체계(웹푸시+스케줄러)**: 토글이 실제로 작동하게. 체감 효과 가장 큼.
2. **P0 — 권한 안전장치 + 감사 로그**: 매니저 위임을 안심하고 쓰게.
3. **P0 — 계정삭제/개인정보 관리 화면**: 정책·스토어 등록 필수.
4. **P1 — 매니저 직접 초대 + 다매장 통합 대시보드**: 다매장 원격관리 핵심.
5. **P2 — 더보기 보강, 매장별 매니저 관리, 권한 세분화, 계정 보안 보강.**

원하시는 우선순위를 알려주시면 P0부터 순차 구현하겠습니다.
